# devops-capstone-project

## Description

This project is a DevOps Capstone Project focused on building a Customer Accounts microservice. The service provides RESTful APIs to create, read, update, and delete customer account information.

The project demonstrates key DevOps and software development practices, including:
- Microservices architecture
- REST API development
- Database integration
- Version control using Git
- Continuous integration and deployment (CI/CD)

The goal is to design, develop, and deploy a fully functional and scalable customer account service.
